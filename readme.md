# SocialLFG

## Purpose

SocialLFG is a World of Warcraft addon that enhances the social experience by integrating Looking For Group (LFG) functionality with guild and friend management. It helps players find groups for dungeons, raids, and other activities while leveraging their social networks.

## Usage

**Finding Groups**: Open the LFG panel and browse available groups created by friends and guild members

## Development

### Release Process

**Build**: Run `powershell -ExecutionPolicy Bypass -File .\release.ps1 -Version X.X.X` to update version number, commit, tag, and zip the addon
